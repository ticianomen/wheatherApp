# Individual Project - Wheather APP

### Project
Weather APP that consumes API WEATHER to show wheather in real time in different cities

### API

For this project we consumed the Wheather API(http://api.openweathermap.org/)
with a free API Key given by them

### Getting Started

I left the API key inside de code
just run:
npm install

npm start
